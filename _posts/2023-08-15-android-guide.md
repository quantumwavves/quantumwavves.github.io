## ADB (Android Debug Bridge)
**¿Que es?**
<p style="text-align:justify">Es una herramienta de programación utilizada para la depuración de dispositivos basados en Android. El daemon de android se conecta con el servidor de la pc a traves de USB o TCP. Esta herramienta nos permite realizar mover, eliminar, copiar archivos y instalar o desinstalar aplicaciones a traves de una terminal.</p>

## Termux & ADB Tricks

### Termux

#### Simple web server con python.

```python
# 8000 es el puerto y x.x.x.x la ip de nuestro dispositivo
python -m http.server 8000 -b x.x.x.x
```
#### Para conectarse a nuestro dispositivo por SSH:

Dependencias necesarias: openssh.

```bash
#Habilitar servicio de ssh en termux
sshd
```

```bash
# Conectarse por ssh
ssh -P 8022 <usuario>@<ip-device>
```
Si desconoces tu usuario y tu ip, puedes consultarlos con los siguientes comandos
{: .success}
```bash
#Mostrar nombre del usuario
whoami
```
```bash
#Listar direcciones ip.
ip a
```
#### SCP de android a pc

Sintaxis:
```bash
# file es el archivo que queremos compartir
# d-user es el usuario de nuestro cliente, en este caso la pc
# d-ip es la ip de nuestro cliente destino.
scp <file> <d-user>@<d-ip>:/Ruta/para/guardar/archivo
```
Ejemplo sencillo:
```bash
scp index.html myuser@192.168.100.24:/home/myuser/Downloads/index.html
```
#### SCP de pc a android

Sintaxis:
```bash
# file es el archivo a compartir
# usuario es nuestro usuario de android
# ip-android, es la ip de nuestro dispositivo
scp -P 8022 <file> <usuario>@<ip-android>:/Ruta/para/guardar/archivo
```
Ejemplo sencillo:
```bash
scp -P 8022 index.html u0_a342@10.10.10.34:/data/data/com.termux/files/home/storage/Downloads
```
#### FTP con Termux.

Dependencias necesarias: busybox termux-services<br>
Termux FTP tiene un login anonimo, cualquiera con acesso a tu red puede hacer uso de este servicio, para colocar un usuario y contraseña usa SFTP(openssh)
{: .warning}

```bash
#habilitar al arranque servicio de ftpd
sv-enable ftpd
# levantar el servicio de ftpd
sv up ftpd
# apagar servicio de ftpd
sv down ftpd
```
Para SFTP:
```bash
#Conectarse al servidor sftp
sftp -P 8022 <ip-android>
```

### ADB
#### Desinstalar aplicaciones.
```bash
#Listar todas las aplicaciones
adb shell pm list packages -s
#Desinstalar aplicaciones
#app-package-name es el nombre de la aplicacion a desinstalar
adb shell pm uninstall -k –user 0 <app-package-name>
```
